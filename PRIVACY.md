# Privacy Policy

Quick Video Speed Control is designed to work locally in your browser.

## Data collection

This extension does not collect, sell, share, or transmit personal information to the developer or to any third-party server.

## Local browser data

The extension stores user preferences with the browser's extension storage, including:

- whether the extension is enabled
- playback speed settings
- shortcut key settings
- seek step settings
- speed badge display settings
- download prompt preference

These settings remain in the user's browser profile or browser sync storage, depending on the browser configuration.

## Page and video access

The extension runs on webpages to detect HTML5 video elements and control playback speed when the user uses the configured shortcuts. It may read the current page title and directly accessible video URL only to show video status, copy a visible video address, or submit a user-requested download to the browser download manager.

The extension does not bypass DRM, login restrictions, blob streams, or segmented streaming formats.

## Downloads

When the user clicks "Save current video", the extension sends the directly accessible video URL to the browser's built-in download manager. The video URL is not sent to the developer.
