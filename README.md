# Quick Video Speed Control

Quick Video Speed Control is a lightweight Microsoft Edge / Chromium browser extension for controlling HTML5 video playback with configurable keyboard shortcuts.

The extension is built for moments when normal playback controls feel too slow: hold a shortcut key to temporarily boost video speed, release it to restore the original playback rate, and adjust the experience from a compact popup or full options page.

Quick Video Speed Control works locally in the browser. It focuses on playback control, convenience, and clear limits rather than attempting to bypass website restrictions.

## Features

- Temporary speed boost while holding a configurable shortcut key.
- Automatic restoration of the original playback rate after release.
- Custom playback speed, trigger delay, shortcut keys, and seek step.
- Keyboard-based forward and backward seeking.
- Optional on-page speed badge with configurable position.
- Popup controls for quick adjustments.
- Full options page for detailed configuration.
- Direct-video saving for browser-accessible MP4, WebM, Ogg, and similar media URLs.
- Clear handling for restricted streams such as DRM, blob URLs, login-protected media, and segmented formats.
- Local preference storage through the browser extension storage API.

## Privacy

Quick Video Speed Control does not collect, sell, share, or transmit personal information to the developer or to third-party servers.

User preferences are stored in the browser. Video URLs are used locally only when showing video status, copying a visible video address, or submitting a user-requested download to the browser download manager.

See [Privacy Policy](PRIVACY.md) for details.

## 中文简介

Quick Video Speed Control 是一款轻量级 Microsoft Edge / Chromium 浏览器扩展，用于通过可自定义快捷键控制 HTML5 视频播放速度。

它适合在观看网页视频时临时提速：按住快捷键进入临时倍速，松开后恢复原来的播放速度，并可通过弹窗或完整设置页调整相关行为。

该扩展在浏览器本地工作，重点是播放控制、便捷调节和清晰的功能边界，不用于绕过网站限制。

## 功能特点

- 按住可自定义快捷键时临时提升视频播放速度。
- 松开快捷键后自动恢复原播放速度。
- 可自定义倍速、触发延迟、快捷键和快进/后退步长。
- 支持键盘快进和后退。
- 可显示页面倍速提示浮层，并调整显示位置。
- 提供弹窗面板用于快速调整。
- 提供完整设置页用于细节配置。
- 支持保存浏览器可直接访问的 MP4、WebM、Ogg 等视频直链。
- 对 DRM、blob URL、登录受限媒体、分片流等受限资源给出清晰提示。
- 使用浏览器扩展存储 API 在本地保存用户偏好设置。

## 隐私

Quick Video Speed Control 不会向开发者或第三方服务器收集、出售、共享或传输个人信息。

用户偏好设置保存在浏览器中。视频地址只会在显示视频状态、复制可见视频地址，或用户主动请求下载时在本地使用。

详情请查看 [隐私政策](PRIVACY.md)。
