# Quick Video Speed Control

播放网页视频时，长按你设置的倍速键临时切换到设定倍速，松开后恢复原来的播放速度。默认倍速键是空格。

## 安装到 Chrome 或 Edge

1. 打开扩展管理页：
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
2. 打开右上角的“开发者模式”。
3. 点击“加载已解压的扩展”。
4. 选择这个文件夹：

   `D:\DirMove\Document\BrowserExtensions\video-space-speed`

## 使用

1. 打开任意视频网站并开始播放。
2. 长按倍速键进入临时倍速，默认是空格。
3. 松开空格恢复原速度。
4. 默认按左方向键后退，右方向键快进；步长可在设置页里调整。
5. 点击浏览器工具栏里的扩展图标，可以快速调整倍速、快进/后退步长、触发延迟、短按倍速键行为，并尝试保存当前视频。
6. 在扩展详情里点击“扩展程序选项”，可以打开完整的可视化设置页。

## 可控制特性

- 插件启用或停用
- 倍速按住键
- 长按时的临时倍速
- 长按触发延迟
- 短按倍速键是否暂停视频
- 快进/后退键
- 快进/后退秒数
- 倍速提示是否显示
- 倍速提示显示位置
- 松开后是否恢复网页原来的播放速度
- 保存当前直链视频
- 保存时是否询问下载位置
- 倍速按住键自定义
- 快进/后退键自定义
- 快进/后退秒数自定义

## 保存视频说明

插件可以保存浏览器能直接访问的 MP4、WebM、Ogg 等视频直链。部分视频网站使用 DRM、blob、M3U8 分片流或登录鉴权，扩展无法绕过这些限制；这种情况下界面会提示“受限流”，并尽量提供复制可见地址的选项。
